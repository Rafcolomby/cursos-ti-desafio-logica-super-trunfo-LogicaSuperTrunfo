# Super Trunfo - Comparação de Cartas (C)

Projeto desenvolvido a partir de requisitos propostos na disciplina de Introdução à Programação.

## Objetivo
Desenvolver um sistema em linguagem C para cadastro e comparação de cartas de cidades, aplicando lógica de programação e estruturas de decisão.

## Funcionalidades implementadas
- Cadastro de cartas com atributos (estado, código, nome da cidade, população, área, PIB e pontos turísticos).
- Comparação entre duas cartas com base em atributo escolhido.
- Determinação automática da carta vencedora.

## Conceitos aplicados
- Estruturas condicionais (if / else)
- Entrada e saída de dados
- Variáveis e tipos primitivos
- Lógica de programação

## Status
Projeto acadêmico concluído.
